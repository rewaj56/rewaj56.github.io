About Me

Hi, I'm Rewaj Gautam, a software developer from Chitwan, Nepal, with a BSc in Computer Science and hands-on experience in IT support and software development. I specialize in building efficient, user-centric systems, particularly in healthcare management.

I started as a Technical Support Intern at Chitwan Medical College, managing IT support for over 2,000 staff. I reduced issue resolution times by 50% and expanded network infrastructure, contributing to the hospital's growth.

Later, as a Software Developer Intern at MEDiiSHA Nepal, I modernized the endoscopy module by revamping the UI with Bootstrap, automating patient detail loading, and implementing dynamic form generation. This improved report creation speed and achieved zero post-implementation issues. I also provided technical support to clients nationwide, troubleshooting via AnyDesk and documenting common issues for knowledge sharing.

Now, as a Software Developer at MEDiiSHA Nepal, I've worked on a real-time queue management system to improve patient flow and reduce confusion about wait times. I also engineered an OPD report system, integrating patient history, diagnosis, and investigations into one unified format. This system syncs with the hospital's billing software to prevent errors and streamline operations. Additionally, I automated late fee management for a college, eliminating manual Excel tracking by providing real-time fee updates.

I'm skilled in PHP, SQL, Laravel, Python, and CSS frameworks like Bootstrap and Tailwind CSS. You can check out my projects on GitHub or connect with me via email. I'm always open to new opportunities and collaborations!

Projects

Kati Kamayoo: Kati Kamayoo is designed to extract detailed product information from Daraz Nepal using Selenium for automation. It gathers data such as product titles, prices, and sales figures, which is then cleaned and stored in a MySQL database. This data is served through Flask and visualized with Chart.js, offering dynamic and interactive insights. The project is open to contributions, and I encourage anyone interested to explore and help enhance its features.
